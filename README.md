# MNIST CNN vs RNN Classification

## Overview
This project implements and compares Convolutional Neural Networks (CNN) and Recurrent Neural Networks (RNN) for handwritten digit classification using the MNIST dataset.

## Features
- MNIST handwritten digit recognition
- CNN implementation using PyTorch
- RNN implementation using PyTorch
- Model training and evaluation
- Accuracy comparison between CNN and RNN
- Confusion Matrix Analysis

## Technologies Used
- Python
- PyTorch
- Torchvision
- NumPy
- Scikit-learn
- Matplotlib

## Dataset
The MNIST dataset contains 70,000 grayscale images of handwritten digits (0–9).

- Training Images: 60,000
- Test Images: 10,000
- Image Size: 28×28 pixels

## Models Implemented

### CNN
- Convolution Layers
- ReLU Activation
- Max Pooling
- Fully Connected Layers

### RNN
- Recurrent Neural Network
- Sequence-based Image Processing
- Hidden State Learning

## Results
The project compares CNN and RNN performance on the MNIST dataset and evaluates their effectiveness for image classification tasks.

## Applications
- Handwritten Digit Recognition
- Optical Character Recognition (OCR)
- Computer Vision
- Deep Learning Research
